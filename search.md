Adapto y refactorizo el código de algunos ejemplos del curso HarvardX:CS50’s Introduction to Artificial Intelligence with Python, y creo un módulo completamente OO para resolver problemas mediante los algoritmos de búsdqueda BFS y DSF


### Script de resolución de laberintos
Utilizando el módulo anterior, este script resuelve laberintos


### Script de resolución de puzzles
Utilizando el módulo anterior, este script resuelve cierto tipo de puzles
